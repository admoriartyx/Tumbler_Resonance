# Mug Resonance Analysis

This project analyzes the resonant frequency of a stainless steel tumbler using physical modeling and FFT-based audio analysis.

## Key Result
- Measured frequency: ~734 Hz (≈ F#5)
- Theoretical prediction (n=2): ~277 Hz
- Theoretical prediction (n=3): ~785 Hz

See `report/` for full write-up and `code/` for analysis.
